# Product Requirements Document (PRD)

## Overview
This document defines the requirements for all sites managed by this PRD propagator.

---

## Site Requirements

### Branding
- **Primary Color**: #your-color-here
- **Secondary Color**: #your-color-here
- **Font Family**: Inter, system-ui, sans-serif

### Pages Required
1. **Home Page**
   - Hero section with headline and CTA
   - Features/services section
   - Testimonials
   - Contact form

2. **About Page**
   - Company story
   - Team section
   - Mission/values

3. **Contact Page**
   - Contact form
   - Location/map
   - Contact information

### Features
- [ ] Mobile responsive design
- [ ] Fast loading (< 3s)
- [ ] SEO optimized
- [ ] Accessible (WCAG 2.1 AA)

### Technical Requirements
- React/Vite stack
- TailwindCSS for styling
- TypeScript for type safety

---

## Site-Specific Customizations

Each site should customize:
- Logo and branding colors
- Contact information
- Business-specific content
- Images and media
